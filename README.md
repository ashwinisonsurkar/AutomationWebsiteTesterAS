# Selenium Automation Framework

## Description
This project automates web application testing using Selenium WebDriver, Java, and TestNG.

## Prerequisites
- Java 17
- Maven
- Chrome Browser

## Installation
1. Clone the repository.
2. Import the project into Eclipse.
3. Run `mvn install`.

## Run Tests
mvn test

## Technologies Used
- Selenium WebDriver
- Java
- TestNG
- Maven
